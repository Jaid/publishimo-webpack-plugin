## Development



Setting up:
```bash
git clone git@github.com:jaid/publishimo-webpack-plugin.git
cd publishimo-webpack-plugin
npm install
```
Testing:
```bash
npm run test:dev
```
Testing in production environment:
```bash
npm run test
```
